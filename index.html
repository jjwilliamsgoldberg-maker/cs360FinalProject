San Francisco Housing Explorer (2007-2024)

An Interactive Dashboard for Exploring SF Housing Prices

By Julian Williams-Goldberg | CS360 Data Visualization | University of San Francisco

Project Description

As a transfer student who arrived at USF without ever visiting San Francisco, I was shocked by the housing prices in the area. This project creates an interactive visualization dashboard that allows users to explore San Francisco housing price data from 2007 to 2024, providing insights into how prices have changed over time across different neighborhoods.

The dashboard enables temporal exploration of price trends, geographic comparison between neighborhoods, analysis by property type (houses, condos, multi-family), and personal context by comparing specific neighborhoods to citywide trends.

Dashboard Features

  • Interactive Choropleth Map: SF neighborhoods colored by median housing price with click-to-select functionality and hover tooltips
  • Time Series Line Chart: Median price trends from 2007-2024 for selected neighborhoods with citywide comparison
  • Ranked Bar Chart: Top neighborhoods by median price, dynamically updated by selected year
  • Scatter Plot: Price vs. square footage for properties in the selected neighborhood, colored by property type (house, condo, multi-family)
  • Filter Controls: Year slider, property type buttons (All/Houses/Condos), and reset functionality

Data Source

Primary Dataset: Assessor Historical Secured Property Tax Rolls
Source: San Francisco Open Data (data.sfgov.org)
Coverage: Fiscal years 2007-2024

Supplementary Data: SF Analysis Neighborhoods GeoJSON boundaries
